### 当前版本号：4.0.0  

## 简介
brick.js 本身是运行在浏览器端的框架，处理 模块加载，模块依赖，页面渲染，路由切换等操作。  
brick.js 需要结合工程化 [node-brick3](http://www.baidu.com) 一起使用。
